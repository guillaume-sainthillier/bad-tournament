    Récupération

    Si vous possédez Git, tapez dans un invité de commande:

    $ cd path/to/www
$ git clone https://github.com/guillaume-sainthillier/bad-tournament.git

    Si vous ne possédez pas Git (ce qui est déconseillé), téléchargez le projet au format Zip ou Tar.gz, et extrayez le dans le dossier www de votre serveur apache.

    Déploiement

    Adaptez les variables $host, $user, $pass et $db du fichier ./connection.php suivant la configuration de votre serveur.
    Importez le fichier bad_tournament.sql dans votre base de données.

    Pré requis

    Vous devez avoir installé:

    • un serveur apache et PHP4.5+
    • une base de données MySQL

    En fait, reportez vous aux installations de :

    • sous Windows : WAMP ou easyphp
    • sous Linux : LAMP

    Si vous débutez, il est plus simple de garder MySQL qui est packagé avec wamp ou easyphp.
